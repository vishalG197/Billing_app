# Billing_app
Streamline expense management with our Billing System App: Create, edit, and track bills effortlessly while gaining valuable sales insights.

-----
### billing website for expenses management* 

frontend https://sales-vishalg197.vercel.app/

backend https://bmi-api-wfcz.onrender.com

## Frontend

HOW TO GO THROUGH THE PROJECT -
- First, visit the link that shows you the landing page, and then go to the add item page and add items.
- after adding the item create a new bill and by selecting item and item-quantity.
- on click of checkout it redirects to the checkout page from where you can create a new bill
- on my bill page you can see your all bills
- on the sales page you are able to see the total revenue generated
